# Virtual Storage (Cloud Storage)



* Often referred to as cloud storage.
* Data can be accessed from any location with internet access.
* Data can be accessed from any location with internet access.
* Data can easily be shared between devices, without the need of having to copy the data and transfer it to and from devices on media such as USB pens.
* Often considered limitless from the point of view of the user as if they need more storage, they can simply purchase some from the cloud.
* It can be quite expensive if you wish to store large amounts of data.
* Access times can be slow in areas with poor internet coverage.
* Physically, the data is stored (hosted) by the company in large data centres, most likely on magnetic disks or SSDs.
* Virtual storage can also refer to the abstraction (separation) of logical storage from physical storage.
* A user could access a virtual storage device without regard to the physical storage structure behind the scenes.
